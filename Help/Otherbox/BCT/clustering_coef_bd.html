SYNOPSIS

function C=clustering_coef_bd(A)

DESCRIPTION 

CLUSTERING_COEF_BD     Clustering coefficient

   C = clustering_coef_bd(A);

   The clustering coefficient is the fraction of triangles around a node
   (equiv. the fraction of node’s neighbors that neighbors of each other).

   Input:      A,      binary directed connection matrix

   Output:     C,      clustering coefficient vector

   Reference: Fagiolo (2007) Phys Rev E 76:026107.


   Mika Rubinov, UNSW, 2007-2010

0018 %Methodological note: In directed graphs, 3 nodes generate up to 8
0019 %triangles (2*2*2 edges). The number of existing triangles is the main
0020 %diagonal of S^3/2. The number of all (in or out) neighbour pairs is
0021 %K(K-1)/2. Each neighbour pair may generate two triangles. "False pairs"
0022 %are i<->j edge pairs (these do not generate triangles). The number of
0023 %false pairs is the main diagonal of A^2.
0024 %Thus the maximum possible number of triangles =
0025 %       = (2 edges)*([ALL PAIRS] - [FALSE PAIRS])
0026 %       = 2 * (K(K-1)/2 - diag(A^2))
0027 %       = K(K-1) - 2(diag(A^2))
0028 
0029 S=A+A.';                    %symmetrized input graph
0030 K=sum(S,2);                 %total degree (in + out)
0031 cyc3=diag(S^3)/2;           %number of 3-cycles (ie. directed triangles)
0032 K(cyc3==0)=inf;             %if no 3-cycles exist, make C=0 (via K=inf)
0033 CYC3=K.*(K-1)-2*diag(A^2);    %number of all possible 3-cycles
0034 C=cyc3./CYC3;               %clustering coefficient
