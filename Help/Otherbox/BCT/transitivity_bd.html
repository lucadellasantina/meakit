TRANSITIVITY_BD    Transitivity

   T = transitivity_bd(A);

   Transitivity is the ratio of 'triangles to triplets' in the network.
   (A classical version of the clustering coefficient).

   Input:      A       binary directed connection matrix

   Output:     T       transitivity scalar

   Reference: Fagiolo (2007) Phys Rev E 76:026107.


   Mika Rubinov, UNSW, 2007-2010

0019 %   Methodological note: In directed graphs, 3 nodes generate up to 8
0020 %   triangles (2*2*2 edges). The number of existing triangles is the main
0021 %   diagonal of S^3/2. The number of all (in or out) neighbour pairs is
0022 %   K(K-1)/2. Each neighbour pair may generate two triangles. "False pairs"
0023 %   are i<->j edge pairs (these do not generate triangles). The number of
0024 %   false pairs is the main diagonal of A^2. Thus the maximum possible
0025 %   number of triangles = (2 edges)*([ALL PAIRS] - [FALSE PAIRS])
0026 %                       = 2 * (K(K-1)/2 - diag(A^2))
0027 %                       = K(K-1) - 2(diag(A^2))
0028 
0029 
0030 S=A+A.';                    %symmetrized input graph
0031 K=sum(S,2);                 %total degree (in + out)
0032 cyc3=diag(S^3)/2;           %number of 3-cycles (ie. directed triangles)
0033 K(cyc3==0)=inf;             %if no 3-cycles exist, make T=0 (via K=inf)
0034 CYC3=K.*(K-1)-2*diag(A^2);    %number of all possible 3-cycles
0035 T=sum(cyc3)./sum(CYC3);        %transitivity
