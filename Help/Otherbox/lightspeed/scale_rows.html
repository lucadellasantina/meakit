scale_rows

PURPOSE

SCALE_ROWS Scale each row of a matrix.

SYNOPSIS

function y = scale_rows(x,s)

DESCRIPTION 

 SCALE_ROWS      Scale each row of a matrix.
 SCALE_ROWS(x,s) returns matrix y, same size as x, such that
 y(i,:) = s(i)*x(i,:)
 It is more efficient than diag(s)*x.
