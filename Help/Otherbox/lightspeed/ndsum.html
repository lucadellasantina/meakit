PURPOSE

NDSUM Multi-dimensional summation.

SYNOPSIS

function x = ndsum(x,dim)

DESCRIPTION 

 NDSUM    Multi-dimensional summation.
 NDSUM(X,DIM) sums out the dimensions in DIM, and squeezes the result.
