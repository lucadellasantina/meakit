sqdist

PURPOSE

SQDIST Squared Euclidean or Mahalanobis distance.

SYNOPSIS

function m = sqdist(p, q, A)

DESCRIPTION 

 SQDIST      Squared Euclidean or Mahalanobis distance.
 SQDIST(p,q)   returns m(i,j) = (p(:,i) - q(:,j))'*(p(:,i) - q(:,j)).
 SQDIST(p,q,A) returns m(i,j) = (p(:,i) - q(:,j))'*A*(p(:,i) - q(:,j)).
