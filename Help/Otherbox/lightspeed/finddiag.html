SYNOPSIS

function i = finddiag(A,k)

DESCRIPTION 

FINDDIAG    Index elements on the diagonal.
 FINDDIAG(A) returns the indices of the diagonal of A.
 FINDDIAG(A,K) returns the indices of the K-th diagonal of A.

 See also DIAG, SETDIAG.
