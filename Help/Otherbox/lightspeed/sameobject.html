DESCRIPTION 

SAMEOBJECT    Object identity
 SAMEOBJECT(A,B) returns 1 if A and B are the same object.
 Specifically, whether their data pointers are the same.

 For example:
 a = 4;
 b = a;
 sameobject(a,b)   % returns 1
 sameobject(a,4)   % returns 0
