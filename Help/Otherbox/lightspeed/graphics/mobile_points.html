mobile_points

PURPOSE

Places n random points in the style given by linespec (e.g. 'bo').

SYNOPSIS

function h = mobile_points(n, linespec)

DESCRIPTION 

 Places n random points in the style given by linespec (e.g. 'bo').
 Points can be dragged with the mouse.
