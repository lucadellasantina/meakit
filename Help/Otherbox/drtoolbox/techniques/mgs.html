DESCRIPTION 

MGS Performs a modified Gram-Schmidt orthogonalization

   [Q, R] = mgs(A)

 Performs a modified Gram-Schmidt orthogonalization. This is a more stable
 way to compute the Gram-Schmidt orthogonalization than Matlab's one.
