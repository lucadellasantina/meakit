symmetricuncertainty

PURPOSE

SYMMETRICUNCERTAINTY symmetric redundancy measure for categorical-discrete

SYNOPSIS

function su = symmetricuncertainty(x,y)

DESCRIPTION 

SYMMETRICUNCERTAINTY symmetric redundancy measure for categorical-discrete 
data
   SU = SYMMETRICUNCERTAINTY(X,Y) returns the symmetric uncertainty between
   the two column vectors X and Y containing categorical measures.

   SYMMETRICUNCERTAINTY [1] computes a symmetric scaled information 
   measure based on mutual information [2][3] normalized to entropies of 
   each variable X and Y

   Example:
   
       x = [1;2;2;2;0;0;1;0;1;2];
       y = [1;2;2;2;2;1;0;2;1;0];
       su = symmetricuncertainty(x,y);
