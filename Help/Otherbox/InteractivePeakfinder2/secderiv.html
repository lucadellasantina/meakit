secderiv

PURPOSE

Second derivative of vector using 3-point central difference.

SYNOPSIS

function d=secderiv(a)

DESCRIPTION 

 Second derivative of vector using 3-point central difference.
 Example: secderiv([0 1 2 3 4 3 2 1 0]) yields [ 0 0 0 0 -2 0 0 0 0]
  T. C. O'Haver, 2006.
