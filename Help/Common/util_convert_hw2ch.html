util_convert_hw2ch

PURPOSE

UTIL_CONVERT_CH2HW Convert hardware ID into channel ID

SYNOPSIS

function [results] = util_convert_hw2ch(input)

DESCRIPTION 

UTIL_CONVERT_CH2HW Convert hardware ID into channel ID
   Accept serials as in 64 channels, return locations in 8*8
   Input:
       results:    Hardware ID (1 ~ 64)
   Output:
       input:      Channel ID (12 ~ 87)
