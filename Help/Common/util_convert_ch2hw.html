util_convert_ch2hw

PURPOSE

UTIL_CONVERT_CH2HW Convert channel ID into hardware ID

SYNOPSIS

function [results] = util_convert_ch2hw(input)

DESCRIPTION 

UTIL_CONVERT_CH2HW Convert channel ID into hardware ID
   Accept locations in 8*8,return serials as in 64 channels
   Input:
       input:      Channel ID (12 ~ 87)
   Output:
       results:    Hardware ID (1 ~ 64)
